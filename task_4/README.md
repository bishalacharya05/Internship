# Implementing a REST API service by creating a blog service in FastAPI
   In this task i have created REST API blog services using FastAPI 
   It allows users to create,read,update, and delete blog posts 
## Tech stack 
   -Python <br>
   -FastAPI <br>
   -SQLAlchemy <br>
   -PostgreSQL <br>
